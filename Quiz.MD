# **Quiz: How the Web Works**

### **Part A — Multiple Choice (Choose the best answer)**

1. What does a web browser do?
   A) Stores webpages permanently
   B) Requests and displays webpages from servers
   C) Creates server-side databases
   D) Encrypts all network traffic automatically

2. What does HTTP stand for?
   A) HyperText Transfer Protocol
   B) Hyperlink Text Transport Program
   C) Hyper Transfer Text Process
   D) Home Transfer Text Protocol

3. Which status code indicates a page was successfully loaded?
   A) 200
   B) 404
   C) 500
   D) 301

4. Which language provides the structure of a webpage?
   A) CSS
   B) HTML
   C) JavaScript
   D) Python

5. Which language controls the style and layout of a webpage?
   A) HTML
   B) CSS
   C) HTTP
   D) SQL

---

### **Part B — True / False**

6. HTTPS is the secure version of HTTP. (True / False)
7. CSS changes the content of a webpage. (True / False)
8. Every webpage load involves at least one request and one response. (True / False)
9. HTML elements include headings, paragraphs, and links. (True / False)
10. The browser assembles HTML, CSS, images, and scripts to render the page. (True / False)

---

### **Part C — Short Answer**

11. Explain in your own words what happens when you type a URL and press Enter in a browser.

12. Give two examples of HTML elements and describe their purpose.

13. Name one reason why HTTPS is important.

14. What are the main steps a browser takes to render a webpage?

---

### **Answer Key (Trainer Use)**

**Multiple Choice:**
1 → B
2 → A
3 → A
4 → B
5 → B

**True / False:**
6 → True
7 → False
8 → True
9 → True
10 → True

**Short Answer Examples:**
11 → Browser sends HTTP request to server, server responds, browser loads and renders page.
12 → `<h1>` → heading; `<p>` → paragraph
13 → Encrypts communication / protects against eavesdropping
14 → Requests HTML → parses → requests additional resources → renders final page
