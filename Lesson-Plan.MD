# **Lesson Plan: How the Web Works (Intro Concepts)**

**Grade / Level:** Beginner / Intro to Web
**Duration:** 45–50 minutes
**Objective:** Students will understand the basic concepts of how the web works, including browsers, servers, HTTP, HTML, CSS, and how pages load.

---

## **Materials Needed**

* Projector or screen for slides
* Web browser (for live demo)
* Optional: simple HTML/CSS files for demonstration

---

## **Lesson Outline**

### **1. Introduction (5 min)**

* Brief discussion: “What happens when you open a website?”
* Introduce the lesson objectives
* Show Slide 1: **The Web & the Browser**

  * Explain client vs. server
  * Highlight the role of URLs

---

### **2. Request / Response Cycle (10 min)**

* Show Slide 2: **The Request / Response Cycle**
* Explain HTTP requests and responses
* Discuss status codes (**200**, **404**)
* Introduce HTTPS and why encryption matters
* **Activity / Demo:** Use browser DevTools to inspect a request/response for a simple website

---

### **3. HTML — Structure (7 min)**

* Show Slide 3: **HTML: Structure of the Page**
* Explain HTML elements and tags
* Examples: headings, paragraphs, links, images
* Optional: quick live demo of editing an HTML file and refreshing the browser

---

### **4. CSS — Style & Presentation (7 min)**

* Show Slide 4: **CSS: Style & Presentation**
* Explain how CSS adds color, fonts, spacing, and layout
* Demonstrate how CSS works alongside HTML
* Optional: change a style in DevTools and see immediate effect

---

### **5. How a Page Loads (8 min)**

* Show Slide 5: **How a Page Loads**
* Walk through the process step-by-step:

  1. Browser requests HTML
  2. Finds linked CSS, images, scripts
  3. Renders final page
* **Activity / Demo:** Open a webpage and identify HTML, CSS, images in DevTools

---

### **6. Summary & Review (5 min)**

* Show Summary Slide: **How the Web Works**
* Recap main points:

  * Browser ↔ Server
  * HTTP request/response
  * HTML structure
  * CSS style
  * Page loading process
* Quick Q&A to check understanding

---

### **Optional Extension / Homework**

* Have students create a simple HTML page with a heading, paragraph, link, and CSS styling
* Use browser DevTools to inspect request/response of their page
