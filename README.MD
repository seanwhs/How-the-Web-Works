# **Module: How the Web Works (Intro Concepts)**

### **1. The Web & the Browser**

* What the web is
* Client (browser) vs. server
* Typing a URL → starting a connection

### **2. Request / Response Cycle**

* Browser sends an **HTTP request**
* Server sends an **HTTP response**
* Status codes (200, 404)
* HTTPS for secure communication

### **3. HTML — Structure**

* HTML defines the *content* and *structure* of a page
* Elements, tags, headings, paragraphs, links

### **4. CSS — Presentation**

* CSS controls *style* and *layout*
* Colors, fonts, spacing, positioning
* How CSS works with HTML

### **5. How a Page Loads**

* Browser requests HTML → parses it
* Additional requests for CSS, images, scripts
* Browser combines everything → renders the page
